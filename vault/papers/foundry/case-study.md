# Case Study

## Candidate

Pick one constrained workflow conversion where Foundry can show something concrete before GCC:

- A small nf-core pipeline or module path with representative channel semantics.
- A CWL user-guide workflow that can be summarized and translated into a Galaxy skeleton.
- A paper methods section with a narrow, well-known tool chain.

## Minimum Publishable Slice

1. Run or demonstrate a source summarizer.
2. Validate the summary against a published schema.
3. Use cast skills for at least one downstream Mold.
4. Produce a Galaxy workflow skeleton or step-level artifact.
5. Run gxwf validation and show the structured feedback loop.
6. Trace at least one instruction/reference through cast provenance.

## Open Decision

The strongest case study is probably the one with the best artifact, not the most impressive biology.
